# Sistem Form Alamat Pengiriman

Sistem untuk mengelola alamat pengiriman dengan fitur:
- Form input alamat
- Simpan data ke file JSON
- Preview alamat
- Cetak/print alamat
- Manajemen data tersimpan

## Fitur Utama
1. **Form Input**: Form alamat lengkap dengan validasi
2. **Preview**: Tampilan preview sebelum menyimpan atau mencetak
3. **Simpan ke JSON**: Data disimpan ke file `data-alamat.json`
4. **Print/Export**: Fitur cetak dengan tampilan rapi
5. **Manajemen Data**: Lihat, hapus, dan kelola data alamat
